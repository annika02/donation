Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents, representing the structure of an HTML or XML document as a tree of objects.

How do you select an element from the DOM?

We can use methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName() to select elements.

What is event delegation in the context of the DOM, and why is it useful?

It's a technique where a single event listener is attached to a parent element to manage events for its child elements. It is useful for reducing memory use and handling dynamic elements.

How do you manipulate an element's attributes and styles using the DOM?

We can use element.setAttribute(), element.getAttribute(), or directly modify properties like element.style for styles.
